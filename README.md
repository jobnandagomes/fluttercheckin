# RO1 Check‑in — Sistema de Pré‑Check‑in e Operações de RecepçãoVisão Geral

 RO1 Check‑in é um sistema para otimizar operações de recepção em hotéis, reduzir filas, padronizar atendimento e fortalecer a jornada do hóspede.

 Baseado na abordagem Boutique Nature, combina fluxo operacional com uma interface elegante e eficiente.

## Funcionalidades
    Fila Realtime
    Exceções Operacionais
    Painel Administrativo
    Relatórios Operacionais
    Configurações do Sistema
    Logout
    Arquitetura

### Frontend: 
   Flutter 
   layout vertical
   design Boutique Nature

### Backend:
Firebase (Firestore, Cloud Functions, Auth, Storage).

### Comunicação: 
Streams Firestore e listeners reativos.
